How to Use Excel Integration
Step 1: Export to Excel
  1. Click "Export to Excel" to create an Excel file with all your inventory
  2. Download the Excel file to your computer
  3. Open it in Excel, Google Sheets, or any spreadsheet software
Step 2: Edit Your Inventory
  1. Inventory Sheet: Add/edit vehicles, descriptions, features
  2. Parts Sheets: Manage parts for each make (Ford_Parts, Lincoln_Parts, etc.)
  3. Services Sheet: Update service offerings
Step 3: Update Website
  1. Save your Excel file as "inventory.xlsx"
  2. Place it in your project folder
  3. Click "Import from Excel" to update your website
Automatic Updates

Run this command to automatically update your website when you save the Excel file:

python inventory_manager.py --monitor
Command Line Tools

You can also use these commands in your terminal:

Export to Excel
python inventory_manager.py --export
Import from Excel
python inventory_manager.py --import
Auto-Monitor
python inventory_manager.py --monitor
